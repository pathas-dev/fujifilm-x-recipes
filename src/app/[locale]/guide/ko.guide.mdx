# 가이드

## 웹사이트 개요
> [Henri-Pierre Chavaz](https://hpchavaz-photography.blogspot.com/) 가 정리한 [레시피 데이터베이스](https://docs.google.com/spreadsheets/d/11PvZ7rOnwmbz5VylVIvzOuWwEa83c0DweAlBwJ2pZJc/edit#gid=1775432829)를 이용해서
> 다양한 레시피를 편리하게 검색해서 볼 수 있도록 하고 개인적으로 사용하는 레시피들을 메모 형식으로 관리할 수 있도록 만든 웹사이트입니다.


## 일러두기
- 1. 로컬 데이터  
데이터베이스가 연동되어 있지 않고 브라우저의 로컬스토리지를 사용하기 때문에 '북마크'나 '커스텀' 페이지 등에서 저장한 데이터가 브라우저 간에 공유되지 않습니다.
- 2. 반응형 디자인  
반응형 디자인을 적용해서 PC 및 모바일 기기에서 최적화된 상태로 사용하실 수 있습니다.
- 3. PWA  
Progressive Web App 기능이 적용되어 있어서 웹사이트에서 설치 후 PC 또는 모바일 기기에서 앱처럼 사용하실 수도 있습니다.

## 페이지 별 가이드

### 북마크 페이지
'레시피' 페이지에서 레시피 카드 우측 상단의 아이콘을 클릭한 카드들을 모아볼 수 있는 페이지입니다.  
카드 우측 상단의 아이콘을 눌러서 북마크를 해제할 수 있으며 해당 페이지를 벗어나기 전에 다시 클릭하면 북마크를 해제하지 않을 수 있습니다.  
페이지 상단의 필터(필름, 카메라, 센서 버튼), 정렬(4번째 버튼), 흑백 토글 버튼으로 카드를 모아서 볼 수 있으며
**이 필터 기능은 '레시피', '커스텀' 페이지에서도 동일한 방식으로 사용할 수 있습니다.**

### 레시피 페이지
데이터베이스에 정리된 레시피들을 카드 형태로 조회할 수 있는 페이지입니다.  
레시피 데이터 자체는 서버에서 가져오는 것이 아니지만
카드마다 이미지 파일을 불러오기 때문에 요금제의 데이터가 충분하지 않은 경우 정렬을 먼저 하신 뒤에 스크롤을 내리시는 것을 추천드립니다.
레시피 카드 우측 상단의 북마크 아이콘을 누르시면 '북마크' 페이지에서 조회 가능하며
카메라 이름을 클릭하면 세부 설정을 볼 수 있습니다.

### 커스텀 페이지
나만의 레시피 설정을 메모해 둘 수 있는 페이지입니다.  
레시피 이름, 카메라, 바탕 필름을 선택한 뒤 바로 아래의 메뉴들을 좌우로 스크롤해서 설정을 조절할 수 있습니다.
설정 조절 후 우측 상단의 '+ 문서' 아이콘을 클릭하면 로컬 스토리지에 레시피 데이터를 저장합니다.
'메일' 아이콘 클릭 시 이메일 입력 후 현재 저장된 레시피들을 JSON 파일로 받아볼 수 있으며
'화살표' 아이콘을 클릭한 뒤 JSON 파일을 등록하면 이메일로 전송한 레시피들을 불러올 수 있습니다.

### 카메라 페이지
후지필름 카메라들이 정리되어 있는 페이지이며 상단의 필름 시뮬레이션 배지로 해당 시뮬레이션을 지원하는 카메라들을 모아볼 수 있습니다.

### 오리진 페이지
데이터베이스를 정리한 Henri-Pierre Chavaz 와 레시피를 만든이들의 목록이 있는 페이지입니다.
