# v1.1.0

## 기능 추가

### 커스텀 레시피 공유 기능
커스텀 레시피 카드 확장 버튼(↓) 클릭 후 하단 클립보드 버튼 클릭 시 공유용 URL 이 클립보드에 복사됩니다.  
공유용 URL 을 눌러 현재 사이트로 방문하게 되면 공유 레시피가 자동으로 커스텀 레시피 목록에 추가됩니다.

### 패치 노트 페이지
버전에 따라 추가된 기능이나 고쳐진 버그 등에 대한 내용을 기재하는 패치 노트를 추가했습니다.

# v1.1.1

## 버그 픽스

# v2.0.0

## 주요 기능 추가

### 🤖 AI 큐레이터 (Chatbot)
인공지능 기반 필름 레시피 추천 시스템이 추가되었습니다.

#### 핵심 기능
- **자연어 질문**: "겨울 느낌 나는 레시피 추천해줘", "영화 같은 분위기 레시피 알려줘" 등 자연스러운 한국어로 질문 가능
- **맞춤형 추천**: 카메라 모델, 촬영 상황, 원하는 분위기를 고려한 개인화된 레시피 추천
- **실시간 이미지 미리보기**: AI가 추천한 설정을 적용한 이미지 보정 결과를 실시간으로 확인
- **기존 레시피 연동**: 데이터베이스의 검증된 레시피들과 연계하여 신뢰성 있는 추천 제공

#### 기술적 특징
- **LangChain + Google Gemini**: 최신 AI 기술을 활용한 자연어 처리
- **벡터 검색**: Pinecone을 통한 의미 기반 레시피 검색
- **이미지 처리**: Sharp를 활용한 실시간 필름 시뮬레이션 적용
- **구조화된 응답**: Zod 스키마를 통한 일관된 응답 형식

#### 사용 방법
1. 네비게이션에서 "AI 큐레이터" 메뉴 선택
2. 원하는 레시피에 대해 자연어로 질문 입력
3. AI가 분석한 맞춤형 레시피와 이미지 미리보기 확인
4. 마음에 드는 설정을 커스텀 레시피로 저장

#### 지원 언어
- 한국어: 자연스러운 한국어 질문과 응답
- 영어: 글로벌 사용자를 위한 영어 지원
